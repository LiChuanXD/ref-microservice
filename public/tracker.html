Exercise Tracker Microservice








Example Usage :

To create a new user =

  1. Enter your User Name in the input field
  2. It will generate a short code for you to use for adding new exercise

To add exercise =

  1. User your new short generated code for adding exercise
  2. Fill in all the input fields, all the data will be stored in database.

To check user's log =

Format :

GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

  1. use the format above to check your user's log
  2. example : url/api/exercise/log?userId=gXop159a&from=2019-12-12&to=2019-12-20&limit=2
  3. from = get the log from this date onwards
  4. to = get the log until this date
  5. limit = show only [n] logs
